# Integration Notes

The C++ module found in legacy versions is now being phased into a Python/MediaPipe pipeline for easier integration with embedded TTS and haptic systems.

## Calibration
Ensure users are 1.5m - 3.0m from the sensor for optimal landmark extraction.

## Performance
- Jetson Nano: ~15 FPS
- Raspberry Pi 4 (4GB): ~10 FPS
- Modern PC: 30+ FPS
